## Data Storage

**File Storage:**

## Maya Integration

**Maya API Usage:**
- Optional conditional import pattern in `src/maya_grass_gen/terrain.py`, `src/maya_grass_gen/generator.py`, and `src/maya_grass_gen/__init__.py`
- Graceful fallback when Maya not available (test/standalone use)
- Maya modules accessed via try/except blocks:
  - `maya.cmds` - Scene manipulation and mesh queries
  - `maya.OpenMaya` - Low-level Maya API (legacy, used for mesh data in `TerrainAnalyzer`)
  - `mash` - MASH network creation (imported dynamically in `src/maya_grass_gen/generator.py`)
- Generated MEL expressions: `WindField.generate_maya_expression()` outputs Maya expressions for wind animation in `src/maya_grass_gen/wind.py`
